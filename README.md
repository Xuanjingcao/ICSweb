# ICS iChangsha

ICS iChangsha 是面向长沙旅客的中文展览旅行指南静态站第一版。

## 本地预览

当前可用本地地址：

```bash
http://localhost:4174
```

如需重新启动：

```bash
python3 -m http.server 4174
```

## 第一版包含

- 首页今日可看、热门入口和城市视觉首屏
- 展览浏览、搜索、日期筛选、区域筛选、类型筛选、免费/收费筛选
- 展览详情弹窗
- 本周日历视图
- 静态地图与推荐路线
- 浏览器本地收藏
- 移动端响应式布局
- imagegen 生成的艺术策展风首页视觉资产

## 数据说明

`app.js` 内置静态演示数据，参考了公开展馆和文旅信息。展期、开放时间、票价、预约方式需以后续场馆官方公告为准。

## 视觉资产

`assets/ics-art-direction.png` 是为 ICS iChangsha 生成的艺术策展风方向图，用作首页主视觉和整体风格参考。
